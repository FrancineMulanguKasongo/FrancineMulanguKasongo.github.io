# Reflection on Using GitHub Pages

## Situation
As part of my PRP370 digital portfolio assessment, I needed to deploy my portfolio using GitHub Pages to showcase my work professionally online.

## Task
I had to configure GitHub Pages to publish my repository as a live website, ensuring all my files (CV, mock interview video, reflections) were accessible to potential employers.

## Action
- I navigated to my repository Settings
- I located the Pages section in the left sidebar
- I selected "Deploy from a branch" as the source
- I chose the main branch and root folder (/)
- I saved the configuration and waited for deployment
- I tested the live site at francinemulangukasongo.github.io
- I verified all links and files were accessible
- I learned how GitHub Pages converts Markdown to HTML

## Result
My portfolio is now live and publicly accessible at https://francinemulangukasongo.github.io/. I learned how easy it is to host a professional portfolio using GitHub Pages. This is a valuable skill for showcasing IT projects to employers. I can now use GitHub Pages for future projects and professional branding.
